# Sandbox UI images (resources/images/ui)

Copied to the build output as resources/images/ui/.

| File | Use |
|------|-----|
| miniwindow.png | MiniWindow 9-slice frame (title bar + body) |
| miniwindow_buttons.png | 14×14 sprite sheet: 10 columns × 3 rows (normal, hover, pressed) |
| panel_side.png | Optional 9-slice for floating [[Panel]] windows |
| icons/analyzerExp.png | EXP analyzer title-bar icon |

### miniwindow_buttons.png layout

- Cell size: 14×14 px
- Rows: 0 = normal, 1 = hover, 2 = pressed
- Columns: 0 = minimize, 2 = close, 7 = lock, 8 = unlock
